Economy (Country, Inflation, ...)

Mit welcher/n Anfrage(n) wird die durchschnittliche Inflation korrekt ermittelt?

  1. SELECT AVG(Inflation) FROM Economy
  2. SELECT SUM(Inflation) / COUNT(Inflation) FROM Economy
  3. SELECT SUM(1.0*Inflation) / COUNT(Inflation) FROM Economy
  4. SELECT SUM(Inflation) / COUNT(*) FROM Economy
