Wie groß ist Europa? 

SELECT SUM(E.Percentage/100 * C.Area)
FROM Encompasses E JOIN Country C 
ON (E.Country = C.Code)
WHERE E.Continent = 'Europe'

Welche Länder sind größer als Europa?

SELECT * 
FROM Country
WHERE Area > (
  SELECT SUM(E.Percentage/100 * C.Area)
  FROM Encompasses E JOIN Country C 
  ON (E.Country = C.Code)
  WHERE E.Continent = 'Europe')
