CREATE TABLE Stundenplan(
  Tag DATE NOT NULL,
  Von TIME NOT NULL,
  Bis TIME NOT NULL,
  Raum VARCHAR(10) NOT NULL,
  Plaetze INT,
  Typ VARCHAR(1),
  Modul VARCHAR(25),
  ModulVerant VARCHAR(20),
  Dozent VARCHAR(20),
  PRIMARY KEY (Tag, Von, Raum)
);

Warum ist die View InHoersaal
nicht automatisch aktualisierbar
(UPDATE-able)?

CREATE VIEW InHoersaal AS
  SELECT Tag, Von, (Bis-Von) AS Dauer,
         Raum, Modul, Dozent
  FROM Stundenplan
  WHERE Typ = 'H';


  1. Die View enthält nicht alle Attribute des Primärschlüssels der darunter liegenden Basis-Relation.
  2. Die View enthält nicht alle NOT-NULL-Attribute der darunter liegenden Basis-Relation.
  3. Das Attribut Typ fehlt in der View.
  4. Die View enthält eine Aggregation von Daten.
